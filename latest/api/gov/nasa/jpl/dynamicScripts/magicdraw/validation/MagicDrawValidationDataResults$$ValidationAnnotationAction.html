ValidationAnnotationAction

Related Doc: package MagicDrawValidationDataResults

abstract class ValidationAnnotationAction extends NMAction with AnnotationAction

In the MD validation result window, multiple annotation actions can be invoked from selected annotations if: - all of the annotation actions have the same NMAction.getID - all the annotation actions implement the MD AnnotationAction interface trait
